Over deze website

Doel

Deze website documenteert het ontwikkelproces van een digitaal prototype binnen het opleidingsonderdeel Prototyping Tools 1. Waar de weekpagina’s het proces chronologisch tonen, focust deze pagina op de opbouw en realisatie van de website zelf.

Ontwerp & structuur

Het traject startte met het uitwerken van de structuur en interface in Figma. Daarbij werden bewuste keuzes gemaakt rond informatiehiërarchie, navigatie en lay-out. Deze ontwerpen vormden de basis voor de technische uitwerking in Visual Studio Code.

Technische uitwerking

De website werd opgebouwd met HTML en CSS, aangevuld met Tailwind CSS voor consistente styling en herbruikbare componenten. Interactieve elementen zoals navigatie, dropdownmenu’s en taalkeuze zijn gerealiseerd met JavaScript.

Contact

Voor de contactfunctionaliteit werd Formspree geïntegreerd, zodat gebruikers berichten kunnen verzenden via een werkend formulier.

Eindresultaat

Het eindresultaat is een overzichtelijke, meertalige website waarin structuur, leesbaarheid en technische samenhang centraal staan.

Objectif

Ce site présente le processus de développement d’un prototype numérique réalisé dans le cadre du cours Prototyping Tools 1. Alors que les pages hebdomadaires montrent l’évolution du projet, cette page se concentre sur la conception et la réalisation du site web.

Conception & structure

Le projet a commencé par la création de la structure et de l’interface dans Figma. Les choix portaient notamment sur la hiérarchie de l’information, la navigation et la mise en page. Ces bases ont ensuite été traduites en une version fonctionnelle via Visual Studio Code.

Réalisation technique

Le site a été développé avec HTML et CSS, complétés par Tailwind CSS pour assurer une mise en forme cohérente. Les éléments interactifs ( navigation, menus déroulants et choix de langue ) ont été implémentés en JavaScript.

Contact

La page de contact utilise Formspree, permettant l’envoi effectif de messages via un formulaire fonctionnel.

Résultat final

Le résultat est un site web clair et multilingue, axé sur la structure, la lisibilité et la cohérence technique.

Ziel

Diese Website dokumentiert den Entwicklungsprozess eines digitalen Prototyps im Rahmen des Kurses Prototyping Tools 1. Während die Wochenseiten den Projektverlauf zeigen, liegt der Fokus dieser Seite auf der Umsetzung der Website selbst.

Design & Struktur

Der Prozess begann mit der Gestaltung von Struktur und Interface in Figma. Dabei standen Entscheidungen zu Informationshierarchie, Navigation und Layout im Mittelpunkt. Diese Entwürfe wurden anschließend technisch in Visual Studio Code umgesetzt.

Technische Umsetzung

Die Website basiert auf HTML und CSS, ergänzt durch Tailwind CSS für ein konsistentes Erscheinungsbild. Interaktive Funktionen wie Navigation, Dropdown-Menüs und Sprachauswahl wurden mit JavaScript realisiert.

Kontakt

Für das Kontaktformular wurde Formspree integriert, sodass Nachrichten zuverlässig über ein funktionierendes Formular übermittelt werden können.

Endergebnis

Das Ergebnis ist eine übersichtliche, mehrsprachige Website, die Struktur, Lesbarkeit und technische Klarheit vereint.
